APPGEO Follow-up Dashboard

PUBLIC GITHUB FILES:
- index.html
- data.js
- manifest.webmanifest
- service-worker.js

PRIVATE ADMIN TOOL:
- APPGEO_Followup_Excel_Updater.html

Initial upload:
Upload all four public files to a new GitHub repository and enable GitHub Pages.

Future Excel updates:
1. Open the private updater.
2. Select the updated Follow-up Excel.
3. Generate data.js.
4. Replace only data.js in GitHub.
5. Commit and refresh.

Password columns are excluded automatically.
